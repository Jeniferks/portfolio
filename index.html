CSS

É a linguagem utilizada para fazer a estilizaçao de um documento HTML. Descreve como os elementos deve ser exibidos.

Javascript

Uma das linguagens de programação mais popular do mundo. Mais conhecida como a linguagem de script para páginas Web

Bootstrap

Framework de front-end utilizado para o desenvolvimento web mais rápido e fácil.

Git e Github

React
